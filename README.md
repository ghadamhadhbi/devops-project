# DevOps FastAPI Project

A task management REST API demonstrating DevOps practices including CI/CD, containerization, Kubernetes deployment, observability, and security scanning.

## Project Status

In Development

## Features
- RESTful API for task management
- Docker containerization
- Kubernetes deployment
- CI/CD pipeline with GitHub Actions
- Observability (metrics, logs, tracing)
- Security scanning (SAST + DAST)

## Tech Stack

- Python 3.11
- FastAPI
- Docker
- Kubernetes
- Prometheus
- GitHub Actions
